# Description

A Go-powererd 'Events Booking' REST API

# Routes

Must support the following routes:

GET         /events
GET         /events/<id>
POST        /events
PUT         /events/<id>
DELETE      /events/<id>
POST        /signup
POST        /login
POST        /events/<id>/register
DELETE      /events/<id>/register

# Stack

- Gotlang standard library
- Gin
